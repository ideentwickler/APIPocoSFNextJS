## Description
An API based on "NestJS" to fetch all relevant data from Salesforce to serve them in all necessary Endpoints for the "POCOApp". Still experimental.

## Installation

```bash
# yarn
$ yarn

# docker
$ docker-compose build
```

## Running the app

```bash
# development yarn
$ yarn start:dev

# docker
$ docker-compose up -d
```
